#  PreEntrega - Placas de Video

Buenas, Este es mi proyecto para la pre-entrega. Armé una página web para vender placas de video

## 🛠️ Armado
* **HTML:** Para armar el esqueleto de la página y acomodar los textos.
* **CSS:** Para darle color, armar el diseño premium del menú y que las fotos no se rompan.
* **Google Fonts:** Para cambiar la letra por una más linda (use la fuente Roboto).
* **Formspree:** Para que el formulario de contacto funcione de verdad y los mensajes me lleguen al mail.
Sume tambien el formspree y use pages 

## 📐 Organización
1. **El Menú:** Quedó fijo arriba con un fondo oscuro. Si bajas en la página, el menú queda inmovilizado.
2. **Los Productos:** Puse tres placas de video con sus fotos y precios. Use Flexbox para que si entrás desde otro dispositivo se pueda ver bien
3. **Las Reseñas:** Los comentarios de los clientes los acomodan. Se adaptan tambien
4. **Contacto:** El formulario para que me manden consultas, vinculado con Formspree, para ver si me llega alguna consulta por mail
5. **En línea:** Ya lo subí a GitHub Pages y quedó la página andando en el hosting de Git con HTTPS
